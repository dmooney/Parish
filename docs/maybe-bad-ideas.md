# Maybe Bad Ideas

Ideas that might be great or might be terrible. Kept here so they don't get lost.

1. **Pets and animals in general** — Living creatures beyond NPCs: dogs, cats, livestock, wildlife. Could add atmosphere and emergent interactions, or could be a massive scope creep rabbit hole.

2. **Allow using LM Studio (and cloud models)** — Support LM Studio as an alternative local backend alongside Ollama, plus optional cloud model providers. More flexibility for users, but more integration surface to maintain.

3. **Birth and death of NPCs** — NPCs that are born, age, and eventually die. Creates a living, evolving world, but adds significant complexity to persistence, relationships, and narrative continuity.
