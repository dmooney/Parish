# The Blow-In Arc play-test.
#
# A stranger steps off the Bog Road on a spring morning in 1820 and
# starts walking the parish. On day one the publican serves them like
# any stranger — a nod, a brief name, nothing more. By the end of the
# month Padraig has the kettle on before the door closes behind them.
#
# This script walks the player through three weeks of the same daily
# rounds (pub, shop, church, farm) and prints `/standing` four times so
# we can watch the familiarity tiers actually move.
#
# Deterministic time advances only (no LLM needed). Run with:
#   cargo run --bin parish -- --script testing/fixtures/blow_in_arc_playtest.txt
